# Fipi Expression And Material Stage Plan

Статус: next reference stage  
Дата: 2026-05-15

## Цель

Подготовить reference package, которого не хватало для manual sculpt:

1. expression sheet;
2. material closeups;
3. proportion guide;
4. final reference approval notes.


Draft-approved turnaround:

```text

```

Character bible:

```text
FP Lead Agent/cg/fipi-master-character/FIPI_CHARACTER_BIBLE.md
```

## Expression Sheet Requirements

Нужно подготовить:

- neutral friendly smile;
- canonical joyful open smile;
- closed-mouth smile;
- blink;
- eyes looking left/right/up/down;
- mild surprised/open-mouth only if future scenes require it.

Сохранять:

- eye shape;
- eye spacing;
- glossy pupils and highlights;
- small glossy black nose;
- green face without pale mask;
- soft cheeks;
- short curved brows;
- non-human mascot identity.

## Material Closeup Requirements

Нужно подготовить closeups для:

- body green plush;
- belly light green plush;
- spines blue plush/soft strands;
- glossy eyes;
- glossy black nose;
- mouth cavity;
- tongue;
- inner ear;
- hands/fingers;
- feet/toes.

## Proportion Guide Requirements

Нужно зафиксировать:

- overall height ratio;
- head/body volume;
- belly patch size and position;
- eye size and spacing;
- nose size;
- mouth width and vertical placement;
- ear diameter and placement;
- arm length/thickness;
- hand/finger proportions;
- leg/foot/toe proportions;
- spine row size/density/direction.

после approval можно запускать manual sculpt production.
